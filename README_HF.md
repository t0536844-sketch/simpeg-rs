# SIM Kepegawaian — RSUD Mimika

Sistem Informasi Manajemen Kepegawaian untuk RSUD Mimika. Aplikasi web berbasis PHP untuk mengelola data pegawai, dokumen, laporan, dan audit trail.

## ✨ Fitur

- 🔐 **Autentikasi & Role** — 3 role (Admin, Operator, Viewer)
- 👥 **CRUD Pegawai** — Kelola data 40+ kolom termasuk dokumen
- 📊 **Dashboard** — Statistik + grafik Chart.js
- 📤 **Import/Export CSV** — Import data dari CSV, export laporan
- 📝 **Audit Trail** — Log aktivitas lengkap + IP address
- 🌙 **Dark Mode** — Toggle tema gelap/terang
- 📱 **Responsive** — Mobile-friendly dengan Bootstrap 5
- 🔒 **Secure** — bcrypt, CSRF, XSS protection, session timeout
